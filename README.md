# Minimal example

## Installation

[Download AVR-GCC for Windows](https://blog.zakkemble.net/avr-gcc-builds/)

Change "WINAVR_PATH" in "make.bat" to point to your winavr bin directory

## Building

### Compile

```
make.bat all
```

### Flash to AVR

```
make.bat flash
```

### Clean

```
make.bat clean
```
